ZendServer-CE

  • Zend server , by default listen on port 10088 change this on /opt/zend/apache2/conf/httpd.conf
  • for administration go to http://127.0.0.1:10081
  • you can run zend server in zend way by calling :
    /opt/zend/bin/zendctl.sh start
    or arch way :
    /etc/rc.d/zendserverce start
    also you can add zenserverce to your daemons in rc.conf
